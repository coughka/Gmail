2.1 How to create a message?

Open mail.google.com.
Click Compose label in the top left corner.
Enter address of the recipient and subject. Type another address to add 2 or more recipients
Compose the message
Click to shift to fullscrean mode.
Click A to format the message. You can change font, font size, font style, font color, text alignment, add ordered and unordered lists or quotes.
Click to add links.
Click face icon to add images.
Click clip icon to add an attachment. For security matters the maximum size of an attachment is 25MB, the bigger files are saved to «Google Drive». See Details
Click dots to check extra options (Labels, Spell Check, Print).
Click Send to send email
Messages are automatically saved in Draft, so you can leave and finish them later. Click bin icon if you don`t want to save the message in Draft
Click Undo to cancel sending an email. Be aware you have only 30 seconds.
Click to turn Confidential Mode (Recipients will not have the option to forward email contents, copy/paste, download, nor print)
